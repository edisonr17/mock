¿Qué es Ruido App Lite?

Es una herramienta web liviana y enfocada en un solo objetivo: tomar la plantilla estándar con los datos de medición ya consolidados y generar el informe oficial 24-001-A30_ listo para entregar, sin copiar datos a mano ni tocar fórmulas directamente.

No requiere base de datos propia. El Excel de salida es el registro oficial. La aplicación es el puente entre la plantilla de datos y el documento final.

¿Cuál es el problema que resuelve? Hoy el técnico consolida los datos de medición en un Excel y luego los copia manualmente al informe oficial, verificando las fórmulas una por una. Con esta herramienta ese proceso se reduce a: subir la plantilla consolidada → revisar → descargar el informe listo. Todo en menos de 20 minutos.
2
¿Cómo funciona?
1
Selecciona dispositivo y profesional

La aplicación carga automáticamente los dispositivos de medición y profesionales disponibles desde la plantilla de configuración almacenada en S3. El técnico los selecciona de un menú desplegable — sin formularios.

2
Sube la plantilla estándar con las mediciones

El técnico sube un único archivo Excel que contiene los datos consolidados: mediciones de fuente encendida, fuente apagada y el tipo de medición correspondiente. La aplicación lo lee automáticamente.

3
Completa los datos del sitio

Dirección, barrio, coordenadas, fechas, horas de medición y los demás campos requeridos por el formato 24-001-A30_. La mayoría ya viene prellenada o se puede copiar de una medición anterior.

4
Dibuja el croquis del lugar

Un lienzo interactivo donde el técnico posiciona los elementos: fuentes de ruido, puntos de medición, barreras, vías. El croquis se exporta como imagen e incluye en el informe.

5
La app genera la plantilla estándar completa

Todos los datos — mediciones de los 3 archivos, información del sitio, dispositivo, profesional y croquis — se insertan en las celdas correspondientes de la plantilla 24-001-A30_. Las fórmulas del Excel hacen el resto.

6
Descarga y listo

El técnico descarga el Excel 24-001-A30_ completamente diligenciado y con cálculos resueltos. Listo para revisar, firmar y entregar. Todo el proceso: menos de 20 minutos.

3
¿Qué se construye exactamente?
Plantilla de configuración (dispositivos y profesionales)

Un archivo Excel almacenado en S3 que contiene la lista de sonómetros disponibles (marca, modelo, serial, fecha de calibración) y los profesionales autorizados (nombre, tarjeta profesional). La aplicación lo lee al iniciar y carga los menús desplegables automáticamente. Para agregar o cambiar un equipo o profesional basta con actualizar ese archivo — sin tocar código.

Lector de la plantilla estándar de mediciones

El técnico sube un único archivo Excel con los datos ya consolidados (fuente encendida, fuente apagada y el tercer tipo de medición). La aplicación detecta automáticamente las columnas, extrae los valores y los mapea a la plantilla de salida.

La consolidación de los archivos del sonómetro en la plantilla estándar se realiza por separado, con una herramienta complementaria que se desarrollará en una fase posterior.
Población del Excel oficial 24-001-A30_

Los datos leídos del sonómetro, junto con los del dispositivo y el profesional, se insertan automáticamente en las celdas correspondientes de la plantilla oficial. Las fórmulas existentes en el Excel hacen el resto: Leq, L90, L50, L10 e incertidumbre se calculan solos.

La plantilla vive en un bucket de Amazon S3. Una función AWS Lambda (Node.js + ExcelJS) la descarga, la llena con los datos y devuelve el archivo al usuario. Para actualizar la plantilla basta con subir el nuevo archivo al bucket — sin tocar código.

Constructor de maqueta / croquis del lugar

Un lienzo interactivo donde el técnico puede armar el plano del sitio arrastrando y soltando elementos: fuentes de ruido, puntos de medición, barreras acústicas, edificios, vías, personas. El croquis resultante se exporta como imagen e incluye en el informe.

Formato de impresión final

Generación del informe listo para entregar: el Excel 24-001-A30_ completamente diligenciado para descarga, y una vista de impresión en PDF con todos los datos, el croquis y la firma del profesional.

Arquitectura técnica
Stack tecnológico
Frontend
  • Vue 3 — interfaz de usuario
  • SheetJS — leer Excel del sonómetro
  • Konva.js — croquis interactivo
  • GitHub Pages — hosting gratuito
Backend
  • AWS Lambda — Node.js
  • ExcelJS — poblar template
  • Amazon S3 — template + archivos generados
  • API Gateway — expone la Lambda
Flujo de datos
① Browser carga plantilla config de S3 → popula selects de dispositivos y profesionales
② Técnico sube plantilla estándar unificada → SheetJS extrae mediciones (FE, FA, 3ro)
③ Técnico completa datos del sitio + dibuja croquis
④ Frontend envía JSON → API Gateway → Lambda
⑤ Lambda descarga template 24-001-A30_ de S3 → llena celdas (ExcelJS)
⑥ Lambda sube Excel generado a S3 → retorna URL firmada
⑦ Browser descarga el informe 24-001-A30_ completo · <20 min todo el proceso
Para actualizar la plantilla Excel basta con subir el nuevo archivo al bucket S3 desde la consola AWS. La Lambda usa automáticamente la versión más reciente sin ningún cambio de código.
